函数

如果实参与形参的类型不同，那么按不同类型数值的赋值规则进行转换。字符型与int型可以互相通用。

在定义函数中指定的形参，在未出现函数调用时，它们并不占内存中的存储单元，在发生函数调用时，函数的形参被临时分配内存单元。

一个函数中可以有一个以上的return语句，执行到哪一个return语句，哪一个return语句起作用。return语句后面的括号可以不要，如return(z);与return z;等价。

过去C标准允许在定义函数时不指定函数类型，此时，编译系统默认它为int型。但是不提倡这样写程序。

函数的首部称为函数原型，需要在使用时先声明。这是C的一个重要特点。

如果在文件开头(主函数外部，include下面)对将要调用的函数声明，这个就会在整个文件内有效。

单个数组元素的传递是按值传递。而传递数组时是按照地址传递。

在定义数组形参时，指定其大小是没有任何作用的，也可以不指定大小，如：

float average(float array[]);

多维数组也可以作为函数参数，可以省略第一维的大小，但不能省略第二维以及其他高维的大小。
