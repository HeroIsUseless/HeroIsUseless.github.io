程序结构

Cy语言中唯一的三目元表达式：表达式1？表达式2:表达式3

表示先计算出表达式1的值为真(非0)，自然直接到表达式2，如果为假(0值)，就绕过表达式2，到表达式3.

switch结构：

switch(表达式)

{

case value1 : 语句1;break;

case value2 : 语句2;break;

...

case valuen : 语句nbreak;

default : 语句n+1

}
