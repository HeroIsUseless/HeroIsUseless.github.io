在文件使用方式中，w,r,a最为基础，w只写，如果指定文件不存在，则自动创建一个新的文件，rb,wb为以二进制进行读写的，ab为向二进制文件中追加数据的，文件不存在就会出错，r+,w+,ab+则代表了可读可写都可以，没有b代表打开一个ASCII文件，有b则代表打开一个二进制文件。出错会返回NULL，它其实被定义为0.它有隐式文档读写标记它会自动移到下一个读写位置，以便读写数据。

计算机在读取ASCII文件时，遇到回车换行符，系统把它转换为一个回车符和一个换行符，但在二进制文件中，不进行这种转换，在内存中的数据与在外部文件中的数据形式完全一致。

其实程序开始运行时系统自动打开3个标准流文件-标准输入流，标准输出流，标准出错输出流，都服务于控制台程序，系统定义了stdin,stdout,stderr，可以通过这3中指针对以上三种流进行操作，它们都以控制台作为输入输出对象。

关闭文件的函数fclose()也包含了一个返回值，成功关闭返回0，否则返回EOF(-1)。

读写函数：fgetc(fp)从fp指向的文件读入一个字符，读成功，带回所读的字符，失败则返回文件结束标志EOF，即-1.

fputc(ch,fp)，向fp中写入一个字符,输出成功，返回值就是输出的字符，输入失败，则返回EOF，即-1.

feof(文件指针)是用来检查指针所指的文件是否结束，如果是，则返回真值1，否则为0

用于读写一个字符串的函数：fgets(str,n,fp)为从fp指向的文件读入一个长度为n-1的字符串，存放到字符串数组str中，读成功，返回地址str，失败返回NULL

fputs(str,fp)把str所指向的字符串写到文件指针变量fp所指向的文件中，输出成功，返回0；否则返回非0值

在向磁盘文件中写入数据时，只输出字符串中的有效字符，并不包括字符串结束标记\0，这样前后两次输入的字符串之间无分隔，连成一片，那么就会无法区分字符串，为了避免出现此情况，在输出一个字符串后，人为输出一个'\n'，作为字符串之间的分隔，即fputs("\n",fp)
