一维数组

C语言不允许对数组大小作动态定义。但在函数中，可以用形参进行动态定义，因为在执行函数时，形参的值时不变的。但是如果指定数组为静态(static)，则不可用可变长数组。

在对全部数组元素赋初值时，由于数组的个数已经确定，因此可以不指定数组长度。例如：

int a[] = {1,2,3,4,5};

在不完全赋值时，在定义数值型数组时，凡未被初始化列表指定初始化的数组元素，系统会自动把它们初始化为0，如果是字符类型，则初始化为'\0'，如果是指针型数组，则初始化为NULL，即空指针。

多维数组

C语言允许使用多维数组。

允许对多维数组进行几行元素的初始化，其他都设为0.

如果对全部元素都赋值，则定义数组时对第一维的长度可以不指定。系统会自动算出第一维长度。在定义时也可以只对部分元素赋初值而省略第一维长度，但应分行赋值，例如：

int a[][4] = {{0,0,3},{},{0,1}}，注意，在大括号后会有逗号。

字符数组：如果提供的初值个数与预定的数组长度相同，在定义时可以省略数组长度，系统会自动根据初值个数确定数组长度。例如：

char[]={'q','w',' ',' w'}

函数

注意scanf对于数组的应用，由于数组的名称就代表它的地址，那么就用它的名称就好了，不用加&，不用加[]，输出也是如此。

可以用putchar函数输出一个字符，用getchar函数输入一个字符，但都必须用回车作为输入标志。注意，getchar可以接受空格，而且是没有参数的，应用返回值获取字符。

printf函数

格式字符:例如%5d，指定输出数据占5列，输出的数据显示在此5列区域的右侧。若输出long(长整型)数据，在格式符d前加字母l（代表long）。若输出long long（双长整型）数据，在各师傅d前加两个字母ll。即%lld

c格式符，用来输出一个字符。一个整数，如果在0~127范围中，也可以用“%c”使之按字符形式输出，在输出前，系统会将该整数作为ASCII码转换成相应的字符。如果整数比较大，则把它最后一个字节的信息以字符形式输出。

s格式符，用来输出一个字符串，如printf（“%s”，“China”）；

f格式符，用来输出实数，不指定输出数据的长度，由系统根据数据的实际情况决定数据所占的列数。系统处理的一般方法是：实数中的整数部分全部输出，小数部分输出6位。指定数据宽度和小数位数，用%m。nf，对最后一位采用四舍五入法。

但是应注意的是，一个双精度数只能保证输出的15位有效的数字的精确度，即使指定小数位数为50，也不能保证输出的50位都是有效的数字。

输出的数据向左对齐，用%-m.nf

e格式符：指定以指数形式输出实数，如果不指定输出数据

注意，如果要输入带有空格的字符串，应该用gets函数，例如：char ch[1000];gets(ch);

所占的宽度和数字部分的小数位数，许多C编译系统（如visual c++）会自动给数字部分的小数位数为6位，指数部分占5列（如e+002），其中e占1列，指数符号占1列，指数占3列，数值按照标准化指数形式输出（即小数点必有而且只用1位非零数字）。所输出的实数共占13列宽度。

格式符e也可以用大写字母E表示，此时输出数据中的指数用大写E表示。

i格式符：作用与d格式符相同，按十进制整型数据的实际长度输出。一般习惯用%d而少用%i

o格式符：以八进制整型形式输出。将内存单元中的各位的值（0或1）按八进制形式输出，因此输出的数值不带符号，即将符号位也一起作为八进制的一部分输出。因此用%o格式声明可以得到存储单元中实际的存储情况。

x格式符：以十六进制数形式输出整数。同样可以用%lx输出长整数，也可以指定输出字段的宽度，如%l2x，用大写字母X时，数据中的字母用大写形式。

u格式符：用来输出无符号型数据，以十进制整型输出

g格式符：用来输出浮点数，系统自动选f格式或e格式输出，选择其中长度较短的格式，不输出无意义的0

如果想输出字符%，用连续两个%表示。

printf("\");这个语句系统会报错。

字符串

在C语言中，是将字符串作为字符数组来处理的。但字符串的特点就是最后有一个8位格式的0：'\0'(空操作符)，C系统在用字符数组存储字符串常量时会自动加一个'\0'作为结束符。

可以用字符串常量来使字符数组初始化，也可以省略花括号。例如：

char c[] = "i am happy."

注意系统自动加\0.

用%c输出一个字符，用%s输出字符串，注意只用数组名，不再加地址符&。可以用scanf("%S",ch);输入一个字符串，注意不支持空格，会自动隔断。

使用字符串处理函数

puts(ch):输出字符串，支持转义字符。在输出时将字符串结束标志'\0'转换成'\n'，即输出字符串后换行。

gets(ch):输入字符串到字符数组，支持空格，包括\0，返回字符数组ch的起始地址。

strcat(ch1,ch2):把ch1和ch2连接起来，结果放在ch1中，返回ch1的地址，因此ch1要足够大。ch1后的\0自动取消。

strcpy(ch1,ch2):将ch2复制到ch1中，如果ch1未初始化，多余部分是未知的，ch2可以是字符串常量。

不能用赋值语句将一个字符串常量或数组直接给一个字符数组。

strncpy(ch1,ch2,n):将ch2的前面n个字符复制到ch1中。一般复制的字符个数n不应该多于ch1中原有的字符(不包括).

strcmp(ch1,ch2):比较ch1和ch2.(按ASCII码对比，遇到\0时停止。如果全部字符相等，则认定两个字符串相等返回0值；若出现不同字符，则以第一对不相同的字符的比较结果为准。若ch1《ch2，输出负整数，若ch1》ch2，返回正整数。)

strlen(ch):测量字符串长度的函数。返回实际长度，不包括\0.也可以直接测试字符串常量的长度。

strupr(ch):转换为大写的函数

注意：在使用字符串处理函数时，应该加头文件#include 《string.h》
