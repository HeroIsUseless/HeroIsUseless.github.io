fromStringToNum()任意输入以空格为间隔的字符串，输出整型数组

为了接受带空格的字符串，应该用gets()函数，但这不是函数的任务，函数仅仅是把字符串转换成整型数组而已。函数返回数组的元素个数。因此除了供输入的字符数组外，还应定义一个供输出的整型数组。

注意：因为在输入时并不能保证空格永远为单个，因此必须对空格进行检测，方法为：定义a1=0,a2=0后，开始检测空格，当检测到一个空格后，就把a2定义到此处之前一个单位，用数字组合函数或者数组输出此数值，然后把a1定义到此位置之后，继续进行检测，但如果第0个，也就是一开头就是空格，就会出现问题，如果第0个是空格，那么a2就会出现错误，是0-1=-1，这是就崩溃了，因此需要另行检测，如果第0个为0，那么就继续检测，直到出现不为0的位置，将a1定义在这里，继续执行检测。

易错点是返回数组必须进行初始化。支持多空格输入，非数字类型输入，首行空格检测，首行非数字检测，检测是否为空格从头开到尾。

最后的一项要单独进行，因为上面的while阻止了\0在下面if中的检测。

下面进行多空格的检测：首先检测字符串内部的，再是开头的，再是结尾的，内部的，遇到空格后，判断下一个是否为空格，如果不是pass掉，如果是，继续判断下一个是否为空格，直到不是空格为止，重新设置计数变量

如果开头为空格，首先应进行if判断，直到不为零的时候出现，定义a1的位置即可，这是一个独立的部分。这时候应该顺便解决了空格从头开到尾的情况

空格直到尾端的情况：也已经解决，但是应该在结尾加一个if判断，因为这时并不能判断是真的结束还是因为空格到头而结束

支持非数字字符隔断：就是空格检测转换成其他字符隔断检测
